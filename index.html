Label-free microscopy

Label-free microscopy has been an essential tool for biology since the invention of phase contrast close to a century ago. However, these techniques have typically required expensive, bulky optical systems.

More recently, jointly designed algorithms and hardware have enabled more flexible, robust label-free imaging systems that access new types of information about samples. For example, shown above is a conventional microscope with its illumination source replaced with a programmable LED array. These techniques are especially well-suited to assays that utilize machine learning.

Using the latest advances in computational microscopy, we can modify a range of existing microscopes to equip them with label-free imaging capabilities. Contact us to learn more about bringing these capabilities to your system.

Microscope control software

We have over a decade of experience building open source tools in multiple programming languages that are used regularly by thousands in the scientific community. For example, Micro-Manager, Pycro-Manager, and Micro-Magellan.

On top of these tools and their thoroughly-tested components, we have built additional layers of application-specific automation for areas such as multi-well plate screening, whole-slide imaging, and 3D culture imaging. These tools can be used to control single microscopes from a local programming environment or graphical user interface, or groups of microscopes in autonomous labs from local networks or the cloud. They handle image data and metadata pipelines so that they can be easily integrated with your own image analysis routines. Finally, they are compatible with hundreds of different microscope hardware configurations, ranging from turnkey systems from major microscope manufacturers to custom-built systems
